玩家常用指令

打开主菜单

这是最核心的指令，所有功能都可以在菜单中完成。

  • /toc cd: 打开玩家导航菜单。

路径点管理

您可以将当前位置保存为路径点，方便以后快速导航回来。

  • /toc nav add <路径点名称>

    • 在您当前的位置创建一个新的路径点。
    • 示例: /toc nav add home

  • /toc nav go <路径点名称>

    • 开始导航至您指定的路径点。
    • 示例: /toc nav go home

  • /toc nav stop

    • 停止当前的导航。

  • /toc nav list

    • 查看所有您已创建的路径点列表。

  • /toc nav remove <路径点名称>

    • 删除一个已创建的路径点。
    • 示例: /toc nav remove old_base

  • /toc nav rename <旧名称> <新名称>

    • 重命名一个路径点。
    • 示例: /toc nav rename home sweet_home

其他指令

  • /toc lang <语言代码>
    • 切换插件的显示语言。
    • 可用语言: zh-CN (简体中文), zh-TW (繁體中文), en-US (英语) 等。
    • 示例: /toc lang en-US

注意事项

  • 本功能不适用于高精度的跑酷寻路。
  • 目前，攀爬类的寻路仅支持梯子和脚手架。
  • 在水下的寻路可能会出现一些意料之外的情况，请谨慎使用。

如果您在使用中遇到任何问题，欢迎向服主&管理员反馈！
